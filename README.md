# BigQuery Data Analysis Project

Projeto de análise de dados utilizando SQL, Python, BigQuery e Google Colab.

## Objetivo
Realizar análise de dados a partir de um modelo dimensional utilizando consultas SQL e processamento em Python.

## Estrutura dos dados

- dim_cliente
- dim_produto
- dim_data
- dim_canal
- fato_venda

## Análises realizadas

- Consultas SQL no BigQuery
- Exploração de dados
- Análise de vendas
- Manipulação de dados com Python

## Tecnologias utilizadas

- SQL (BigQuery)
- Python
- Google Colab
- Pandas
- BigQuery
